 The Twitcherizer is an application that pulls data from two different sources in the Twitch.tv API, and displays a small list of streamers along with some basic information and whether they are online or not, or if their account is closed.

Part of Free Code Camp's "Use the Twitchtv JSON API" challenge, at http://www.freecodecamp.com/challenges/use-the-twitchtv-json-api
